API - Application Programming Interface (Interface de programação de aplicações) FORMA DE UM PROGRAMA SE COMUNICAR COM O OUTRO
GUI - Graphical User Interface (interface grafica de usuario) FORMA DO USUARIO SE COMUNICAR COM O PROGRAMA

HTTP -  Hypertext Transfer Protocol (Protocolo de tranferencia de hypertexto)
    - GET: Requisição de dados
    - POST: Inserir novo dado
    - PUT: Alterar os dados como um todo
    - PATCH: Alterar uma parte dos dados
    - DELETE

STATUS HTTP: 
    Respostas de informação (100-199),
    Respostas de sucesso (200-299), 200 Sucesso
    Redirecionamentos (300-399),
    Erros do cliente (400-499), 404 not found
    Erros do servidor (500-599). 500 Internal Server Error

Arquiteturas de API
SOAP - Simple Object Access Protocol (Protocolo de acesso de objeto simples)
    - XML

REST - REpresentational State Transfer (Transferência Representacional de Estado) ***** Mais utilizada
    - JSON muito parecido com os dicionarios do python
